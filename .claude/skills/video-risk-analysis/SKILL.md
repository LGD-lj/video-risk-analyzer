# 视频风险点分析 Skill

## 目标

本项目用于分析道路/路线视频中的运营风险点，输出时间点、截图、风险等级、风险类型、描述，并生成 Word 报告和截图 ZIP。

重点不是只识别交通事故风险，而是识别对运营路线、无人车通行、测试交付、路线复核有影响的风险。

## 风险点数量原则

风险点数量不是固定 5-10 个。

原则：
- 根据视频实际风险情况输出。
- 风险少可以少于 5 个。
- 风险正常时通常输出 5-10 个代表性点。
- 风险很多时可以输出 10-12 个，但不能重复堆砌。
- 不要为了凑数量输出弱风险。
- 不要过于保守，只输出 2-3 个明显风险。
- 最终目标是数量适中、代表性强、便于人工复核。

## 风险分类

### A 类：强风险，优先保留

- 施工 / 修路
- 施工围挡
- 锥桶 / 临时导流
- 限高 / 低净空 / 桥洞 / 顶棚
- 闸口 / 门岗 / 隔离墩 / 护栏导致通道明显收窄
- 货车 / 工程车明显遮挡或占道
- 路面异常
- 明显需要人工复核或低速通过的点

### B 类：中风险，适量保留

- 非机动车混行
- 行人横穿
- 停车占道
- 窄路
- 出入口密集
- 商铺门口车辆影响通行
- 视线遮挡

### C 类：弱风险，只在明显影响通行时保留

- 普通非机动车路过
- 普通树木遮挡
- 普通路边停车
- 普通商铺门口
- 普通窄路但无明显阻碍
- 短时动态车辆干扰

## 长期风险判断

增加或使用 long_term_risk 字段。

长期风险包括：
- 修路 / 施工围挡 / 长期道路封闭
- 固定限高 / 固定低净空 / 桥洞
- 固定闸口 / 固定门岗 / 固定隔离设施
- 长期占用道路的设施
- 军事基地、园区门岗、固定出入口管制
- 看起来一周以上不会变化的固定障碍或道路条件

不要轻易判断为长期风险：
- 普通非机动车
- 临时停车
- 临时货车
- 临时行人
- 普通树木遮挡
- 普通商铺门口车辆
- 看不出是否长期存在的停车占道

停车占道只有在明显像长期违停、固定占道、连续多个时间段同一位置出现，或结合场景判断可能长期存在时，才写"建议长期关注/后续复核"。

## 去重原则

- 同一施工围挡连续出现，只保留最清晰/最有代表性的一张。
- 同一商铺路段连续停车占道，不要连续输出。
- 同一路段非机动车连续出现，最多保留代表性样例。
- 普通树木遮挡最多保留 1 个，除非遮挡标志、限高牌、闸口、路口视线。
- 同类普通风险使用 60-90 秒去重。
- 强风险使用 20-30 秒去重。
- 时间相近但风险类型明显不同，可以都保留。

## 输出描述风格

每个风险点描述包含：
1. 画面现象
2. 对运营路线/无人车通行的影响
3. 是否可能长期存在
4. 是否建议人工复核

描述不要夸大。如果画面不清晰，用"疑似""建议复核"。

## Word 报告要求

报告要适合运营、测试、交付人员阅读。
重点强调：
- 风险点时间
- 风险等级
- 风险类型
- 画面现象
- 运营影响
- 是否长期风险
- 是否建议复核
